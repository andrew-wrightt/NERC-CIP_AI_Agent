How Retrieval Works

Hybrid Search

Combines two search methods for better accuracy:

  • Vector Search - Understands meaning and context (60%)
  • BM25 Search - Matches exact words and phrases (40%)

Results are combined using Reciprocal Rank Fusion (RRF).

Chunking

  • Documents split into ~1500 character chunks
  • 500 character overlap to preserve context
  • Breaks at paragraph/sentence boundaries when possible

Configuration

Edit RETRIEVAL_CONFIG in server.js to adjust settings.
